# Trade Daily

一个桌面优先的市场复盘工作台。

## 功能概览

- 首页包含 `开始复盘` 和 `市场配置` 两个主入口。
- 支持 `A股`、`美股`、`区块链`、`大宗期货` 四个市场。
- 每个市场首版只配置一个默认品种。
- 点击开始复盘后，并发拉取四个市场的 `240` 根日线 K 线与成交量。
- 代理配置支持开关控制，默认端口 `13004`。
- 前端展示统一的深色图表卡片，后端统一处理跨域、代理和多数据源格式转换。

## 默认数据源

- `A股`：东方财富公开日线接口，默认 `sh000001`
- `美股`：Yahoo Finance v8 chart，默认 `AAPL`
- `区块链`：Binance klines，默认 `BTCUSDT`
- `大宗期货`：新浪期货日线接口，默认 `AU0`

## 本地开发

```bash
npm install
npm run dev
```

- 前端地址：`http://localhost:5173`
- 后端地址：`http://localhost:3001`

## 代理说明

- 默认代理关闭
- 默认代理配置：`127.0.0.1:13004`
- 可在页面的“市场配置”面板中直接开启或关闭

## 常用命令

```bash
npm run dev
npm run check
npm run test
npm run verify:data
```

## 项目结构

```text
src/            前端页面、组件、状态管理
api/            Express 聚合接口与市场 provider
shared/         前后端共享类型与默认配置
scripts/        真实数据拉取验证脚本
```

## 说明

- `npm run verify:data` 会直接通过服务层验证四个默认市场是否能拉到真实数据。
- 若本地访问币安或其他市场需要代理，请在界面中开启代理后再点击开始复盘。
```
